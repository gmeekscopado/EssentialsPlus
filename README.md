# N-Able Partner Portal

MSP-focused partner portal built on Salesforce Experience Cloud.

## Components
- **partnerWelcomeBanner**: Animated welcome banner with partner tier badges
- **securityAlerts**: Real-time security alerts and notifications
- **PartnerPortalController**: Apex backend for data services
